■ゲームの起動
「Game.exe」を実行するとゲームが起動します。


■ゲームの操作
基本的にマウスのみです。
メッセージを次に進めるのは［Enter］でも可能です。


■ゲームの説明
あなたは「案内人」が開催している劇場にやってきたお客さんです。
そこでの催しものとして「人狼ゲーム」を題材にした演目が行われていました。

各演目を見終わった人狼が誰だったのかをあててください。
正解すれば次の演目が見られるようになります。


■人狼の説明
大人数（本ゲームでは11人～17人）で行うパーティーゲームです。
参加者は村人陣営・人狼陣営に分かれます。

村人陣営の目的は村の中にいる人狼をすべて全滅させること。
人狼陣営の目的は村の中にいる村人を全滅させることです。
村人たちは話し合いをすることによって、お互いの情報を出していき
人狼を見つけ出していきます。

このゲームは昼のターンと夜のターンがあります。
昼のターンは話し合いを行い、村から1名を追放します。
夜のターンでは各能力者が能力を行使できます。
また人狼は村人の中から1名を襲撃します。
昼に追放、夜に襲撃された参加者は以降村からいないものとなり話し合いに参加できません。

昼のターンと夜のターンを繰り返して、
人狼がすべていなくなれば「村人陣営」の勝利、
人狼の数と村人の数が同じになれば「人狼陣営」の勝利となります。


■著作権について

このゲームをプレイする以外の用途で使用しないでください。
